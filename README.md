# Project Management App

Django web app to manage requirements and track efforts across projects.

## Development

`uv` is used to manage dependencies and python virtual environments. Follow the official installation guide [here](https://docs.astral.sh/uv/#installation) to get started.

### just

A `justfile` is provided to easily run some common commands. Install `just` with:

```bash
uv tool install rust-just # installs system-wide
```

List the provided commands with:

```bash
just -l
```
